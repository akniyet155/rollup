# Landing Builder

Простой конструктор лендингов на React + Vite + Tailwind.

## Команды

```powershell
cd "c:\Users\ahon\Desktop\Новая папка\landing-builder"
npm install
npm run dev
npm run build
npm run preview
```

## Структура
- `src/blocks`: базовые блоки (Hero, Features, CTA, Footer)
- `src/templates`: шаблоны и метаданные
- `src/pages`: страницы (`Home`, `Builder`)

## Запуск dev-сервера в VS Code
- Откройте палитру команд → `Tasks: Run Task`
- Выберите `Landing: Install` (один раз)
- Затем `Landing: Dev Server`

Откройте [http://localhost:5173](http://localhost:5173) и перейдите на [http://localhost:5173/builder](http://localhost:5173/builder) для конструктора.
